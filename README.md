# ☕ Coffee Product Optimization & Revenue Analysis

This project analyzes product performance for Afficionado Coffee Roasters using transaction-level data. The main goal is to understand which products are popular and which generate the most revenue.

---

## 📌 Objectives

- Identify top-selling and least-selling products  
- Analyze revenue contribution of each product  
- Understand category-level performance  
- Perform Pareto (80/20) analysis  
- Support menu optimization decisions  

---

## 📊 Dataset Details

The dataset includes:

- transaction_id → Unique transaction  
- transaction_qty → Quantity sold  
- unit_price → Price per unit  
- product_category → Category (Coffee, Tea, etc.)  
- product_detail → Product name  
- store_location → Store location  

---

## 🧠 Key Analysis Performed

- Product ranking (Sales & Revenue)  
- Category revenue distribution  
- Revenue contribution (%)  
- Popularity vs Revenue comparison  
- Pareto analysis (Top products driving revenue)  

---

## 📈 Tools Used

- Python  
- Pandas  
- Matplotlib  
- Streamlit  

---

## 🚀 Streamlit Dashboard Features
